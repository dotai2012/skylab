Skylab Demo
===========

Usage
-----

* Install "Expo Go" on your phone
* Clone this repo by these command

```
> git clone https://github.com/dotai2012/skylab.git
> cd skylab
> npm install
> npm run start
```

* The command line will show a QR code, scan this code and the app will open
